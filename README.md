# png_pixelart_resizer
png形式のドット絵をいい感じに拡大して一括出力するWindows用バッチファイルです。

## 特徴

- 複数の入力元フォルダを指定可能（手入力 or ドラッグ&ドロップ）
- 出力先フォルダを1箇所にまとめて指定可能
- 同名ファイルが複数フォルダにあった場合、自動で `(1)`, `(2)` を付けてリネーム
- 保存時にEXIF等のメタデータを自動削除
- [optipng](https://optipng.sourceforge.net/) がある場合、画質・サイズを維持したままファイルサイズをロスレス圧縮

## 動作環境

- Windows 10 / 11
- PowerShell（Windows標準搭載のもので動作します。追加インストール不要）

## ファイル構成

```
resize_nn.bat   ... 実行用バッチファイル（これをダブルクリックして使う）
resize_nn.ps1   ... 実際の処理を行うPowerShellスクリプト（同じフォルダに必須）
```

`resize_nn.bat` と `resize_nn.ps1` は必ず同じフォルダに置いてください。

## 使い方

### 1. 出力先フォルダを設定する

`resize_nn.bat` をテキストエディタで開き、先頭付近の以下の行を、画像を出力したいフォルダに書き換えて保存します。

```bat
set "OUTPUT_DIR=C:\Users\YourName\Desktop\output"
```

### 2. 実行する

`resize_nn.bat` をダブルクリックします。

**入力元フォルダの指定方法は2通りあります。**

- **手入力**: フォルダパスを1つずつ入力し、Enter。すべて入力し終えたら、何も入力せずEnterで入力終了。
- **ドラッグ&ドロップ**: 拡大したい画像が入ったフォルダを（複数選択可）まとめて `resize_nn.bat` にドラッグ&ドロップして実行。

### 3. 拡大倍率を入力する

`2` と入力すれば2倍、`1.5` と入力すれば1.5倍になります。

### 4. 完了

指定した出力先フォルダに、拡大後のPNG画像がまとめて保存されます。

## ファイルサイズ圧縮を有効にする（任意）

[optipng](https://optipng.sourceforge.net/) を使うと、画質・画像サイズを一切変えずにファイルサイズだけを圧縮できます。

1. [optipng公式サイト](https://optipng.sourceforge.net/) からWindows版をダウンロード
2. zipを展開し、中の `optipng.exe` を取り出す
3. `optipng.exe` を `resize_nn.bat` / `resize_nn.ps1` と同じフォルダに配置

配置するだけで、次回実行時から自動的に圧縮が適用されます（未配置の場合は圧縮なしで動作します）。

## 注意事項

- 処理対象は `.png` ファイルのみです
- 元のファイルは変更・削除されません（出力先フォルダに新しいファイルとして保存されます）
- 存在しない入力元フォルダを指定した場合はスキップされます
