# png_pixelart_resizer

指定した複数フォルダ・複数PNGファイルを、**ニアレストネイバー法**（最近傍補間）で指定倍率に一括拡大するWindows用ツールです。ドット絵やアイコンなど、拡大時にぼかしを入れたくない画像に向いています。

## 特徴

- 入力元は「フォルダ」「個別のPNGファイル」どちらも指定可能（混在OK）
- ドラッグ&ドロップで入力元を指定（複数選択・混在可）
- 出力先フォルダを1箇所にまとめて指定可能
- 出力先フォルダ内の既存画像を、処理開始前に削除するかどうかをy/nで選択可能
- 同名ファイルが複数入力元にあった場合、自動で `(1)`, `(2)` を付けてリネーム
- 保存時にEXIF等のメタデータを自動削除
- [optipng](https://optipng.sourceforge.net/) がある場合、画質・サイズを維持したままファイルサイズをロスレス圧縮

## 動作環境

- Windows 10 / 11
- PowerShell（Windows標準搭載のもので動作します。追加インストール不要）

## ファイル構成

```
resize.bat   ... 実行用バッチファイル（これにファイル・フォルダをドラッグ&ドロップして使う）
resize.ps1   ... 実際の処理を行うPowerShellスクリプト（同じフォルダに必須）
```

`resize.bat` と `resize.ps1` は必ず同じフォルダに置いてください。

## 使い方

### 1. 出力先フォルダを設定する

`resize.bat` をテキストエディタで開き、先頭付近の以下の行を、画像を出力したいフォルダに書き換えて保存します。

```bat
set "OUTPUT_DIR=C:\Users\YourName\Desktop\output"
```

### 2. 拡大したいPNGファイル・フォルダをドラッグ&ドロップする

拡大したいPNGファイル、またはPNG画像が入ったフォルダを（複数選択・混在可）まとめて `resize.bat` にドラッグ&ドロップします。

> `resize.bat` をドラッグ&ドロップせずにダブルクリックだけで起動した場合は、ドラッグ&ドロップを促す案内が表示され、何かキーを押すと終了します。

### 3. 出力先フォルダをクリアするか選択する

```
処理開始前に出力先フォルダ内の画像を全て削除しますか？ (y/n):
```

`y` と入力すると、処理開始前に出力先フォルダ内の既存の画像ファイル（png/jpg/jpeg/bmp/gif）がすべて削除されます。`n` またはそれ以外を入力した場合は削除せず、既存ファイルを残したまま追加で出力されます。

### 4. 拡大倍率を入力する

`2` と入力すれば2倍、`1.5` と入力すれば1.5倍になります。

### 5. 完了

指定した出力先フォルダに、拡大後のPNG画像がまとめて保存されます。

## ファイルサイズ圧縮を有効にする（任意）

[optipng](https://optipng.sourceforge.net/) を使うと、画質・画像サイズを一切変えずにファイルサイズだけを圧縮できます。

1. [optipng公式サイト](https://optipng.sourceforge.net/) からWindows版をダウンロード
2. zipを展開し、中の `optipng.exe` を取り出す
3. `optipng.exe` を `resize.bat` / `resize.ps1` と同じフォルダに配置

配置するだけで、次回実行時から自動的に圧縮が適用されます（未配置の場合は圧縮なしで動作します）。

## 注意事項

- 処理対象は `.png` ファイルのみです（ドラッグ&ドロップでそれ以外のファイルを含めた場合はスキップされます）
- 入力元のファイルは変更・削除されません（出力先フォルダに新しいファイルとして保存されます）
- 出力先フォルダのクリア機能は、出力先フォルダ内のファイルを削除します。誤って別の重要なフォルダを出力先に指定しないようご注意ください
